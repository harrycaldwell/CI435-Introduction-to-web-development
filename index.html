Week 4

Published on October 27, 2022

This week in Lectures and Labs we looked at creating the CSS (stylesheet) for the site, this is used to give the website actual structure, we do this by using the language CSS which is a declarative language where we call on things such as:

  1. Class IDs that we have layed out in the HTML code.
  2. Div tags that we have used to divide the HTML pages into sections so that we can later style them individually.
  3. Tags that are used in the code for example h1.

Why do we do this? Well to make the website look pretty, without CSS the site will remain unformatted and there will be no structure to the site at all, so by using CSS we are able to control the structure of the page, in addition to structure it also allows us to customize the site with different fonts of our choice be it a serif or sans serif style font even fonts that are not native to PCs and Mobile devices, we can change the color of backgrounds and control things like where things such as text are placed on the site.

Week 5

Published on November 03, 2022

In this week of labs, we looked at starting to create the page for tutorials, and how to specify font families and a fall back for if the font cannot be displayed correctly, this is done in css by doing the following example:
body{
font-family: georgia, "times new roman", Times, serif;
}

To ensure that text can be correctly displayed on all devices that are able to connect to the website, you should always include system default fonts so that in the event that the font is unable to be retreived from the webserver or fonts source we are still able to display content that should be on the page
